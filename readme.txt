Nyisd meg a Windows Feladatütemezőt. Nyomd meg a Windows + R billentyűkombinációt, majd írd be a taskschd.msc parancsot és nyomd meg az Enter billentyűt.

A Feladatütemező ablakban kattints a "Feladat létrehozása..." lehetőségre a jobb oldali ablaktáblában.

Az "Általános" fülön adj meg egy nevet és egy leírást a feladatnak.

A "Kivitel" fülön válaszd ki, hogy mikor szeretnéd elindítani a feladatot. Például válaszd ki a "Kapcsolódásnál" lehetőséget, hogy a feladat a felhasználó bejelentkezésekor elinduljon.

A "Triggerek" fülön kattints az "Új..." gombra, majd állítsd be a trigger feltételeit (pl. naponta egyszer, vagy bemenetként a rendszer indulásakor).

A "Műveletek" fülön válaszd ki az "Új..." gombot, majd a "Program indítása" lehetőséget.

A "Program/parancsfájl" mezőbe írd be a Python interpreter elérési útvonalát (például C:\Python39\python.exe), majd a "Argumentumok" mezőbe írd be a Python script elérési útját (például C:\Users\Te\Path\To\Your\Script.py).

Kattints az "OK" gombra, majd zárd be a Feladatütemező ablakot.
